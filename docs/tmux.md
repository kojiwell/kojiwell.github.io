## Basics

Start a new Tmux session

```
tmux new -s [session-name]
```

Detach from a Tmux session

```
Ctrl+b, d
```

List Tmux sessions

```
tmux ls
```

Attach a Tmux session

```
tmux a -t [session-name]
```
